# RenatoHUB

Dashboard pessoal unificado para centralizar ferramentas acadêmicas e utilitários do dia a dia. Hospedado via **GitHub Pages**.

## 🧩 Apps Disponíveis

| App | Descrição |
|---|---|
| **Academic Flow** | Controle de presenças e faltas com calendário interativo. Sincronizado com Firebase. |
| **Task Flow** | Agenda de provas e trabalhos com contagem regressiva automática. Sincronizado com Firebase. |
| **Poké Catcher** | Calculadora de catch rates para Pokémon Violet (Gen 9) usando dados da PokéAPI. |

## 🛠️ Stack

- HTML5 + JavaScript puro (sem frameworks)
- [Tailwind CSS](https://tailwindcss.com/) via CDN
- [Lucide Icons](https://lucide.dev/)
- [Firebase Firestore](https://firebase.google.com/) (Academic Flow e Task Flow)
- [PokéAPI](https://pokeapi.co/) (Poké Catcher)

## 📁 Estrutura

```
Renato HUB/

├── academic-flow/
│   └── index.html          — Academic Flow
├── projetos/

└── pokemon-catcher/

```

## 🚀 Como Rodar

Basta abrir `index.html` no navegador ou acessar via GitHub Pages.
